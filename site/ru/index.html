Автономный ИИ-агент, который живёт внутри Telegram

Teleton работает как реальный аккаунт Telegram или бот. Он думает в агентном цикле, помнит диалоги через гибридный RAG и нативно говорит на языке блокчейна TON — отправляет крипту, меняет токены на DEX, участвует в аукционах доменов и проверяет платежи прямо из сообщения в чате.

Запуск за три шага

Нужны только Node.js 20+ и аккаунт Telegram. Без зависимостей сборки.

1

Установка

Возьмите CLI из npm.

npm install -g teleton@latest
2

Настройка

Мастер настроит провайдера, авторизацию Telegram, политики и кошелёк.

teleton setup --ui
3

Запуск

Стартуйте с дашбордом и напишите своему агенту.

teleton start --webui
