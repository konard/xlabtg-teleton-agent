Up and running in three steps

Node.js 20+ and a Telegram account are all you need. No build dependencies.

1

Install

Grab the CLI from npm.

npm install -g teleton@latest
2

Set up

The wizard handles provider, Telegram auth, policies and wallet.

teleton setup --ui
3

Run

Start with the dashboard and message your agent.

teleton start --webui
